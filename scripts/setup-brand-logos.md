# Guide d'installation des logos de marques

Pour ajouter les logos sur la page d'accueil :

## 1. Placez les fichiers dans `public/images/brands/`

Copiez vos fichiers de logos dans le dossier `public/images/brands/` avec ces noms exacts :

- `guess.png`
- `guess-jeans.png`
- `sam-edelman.png`
- `vilebrequin.png`
- `dkny.png`
- `bayton.png`
- `circus-ny.png`
- `gooce.png`
- `pulliez.png`

## 2. Format recommandé

- Format : PNG avec fond transparent
- Hauteur : ~60-80px (largeur proportionnelle)
- Résolution : Haute qualité pour un rendu net

## 3. Vérification

Une fois les fichiers ajoutés, actualisez la page d'accueil. Les logos remplaceront automatiquement le texte.
